# Darwin v3.1 评分体系

> 满分 **150分** = 基础分（17维权重归一化到100分）+ 创造力加分（D16，满分50分）

## 评分公式

```
基础分 = Σ( score_i / max_i × weight_i ) × normalize_factor
normalize_factor = 100 / Σ(weight_i)  其中 i ≠ D16
总分 = 基础分 + D16创造力得分
```

## 计算示例（Python）

```python
dimensions = {
    "D1": {"name": "功能完整性", "weight": 9, "max": 9},
    # ... 全部18维
    "D16": {"name": "创造力", "weight": 0, "max": 50},
}

scores = {
    "D1": 8,  # 各维度得分
    # ...
    "D16": 35,
}

weight_sum = sum(d["weight"] for k, d in dimensions.items() if k != "D16")
nf = 100 / weight_sum

base = sum(scores[k] / d["max"] * d["weight"] * nf 
           for k, d in dimensions.items() if k != "D16")
total = base + scores["D16"]
```

## 常见错误

- ❌ 直接用各维度得分相加（满分不对）
- ❌ 把 D16 创造力算进权重归一化（它是独立加分项）
- ❌ 用 10 分制代替各维度的实际满分（D1-D3 满分9，D4-D8 满分7 等）
